# TaskMaster Brain Dump - Concise Notes

Source file: `docs/source/TaskMaster_brain_dump.pdf` (copied from local Downloads variant `(1)`).

## Core Intent (Distilled)
- Build a highly satisfying daily loop around creating/completing tasks with visible progress.
- Keep the task manager valuable by itself; gamified rewards should amplify, not gate, utility.
- Make input effortless through natural language (including bulk recap updates), then confirm before applying.
- Use AI for personalization and suggestions while preserving clear user control and AI-off usability.
- Emphasize consistency, mindfulness, and balance without guilt-heavy language.

## Repeated Themes Across Raw Prompt / Prompt 2 / Raw Text 3
- Turning thoughts into action (goals -> smaller steps -> daily execution).
- 30-60 second daily review/check-in flow with tomorrow planning.
- Multi-modal capture path (text now; voice recap and photo logging as planned expansions).
- Suggestion feedback loop (`too easy`, `too hard`, `not for me`) to continuously tune relevance.
- Adjustable challenge/personalization intensity ("how much should the app push me?").
- Theme/reward flexibility (minimal vs gamified styles and different reward structures).
